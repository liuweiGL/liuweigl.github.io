小科普：

  1. sources.list：常规的仓库文件，apt-get install 的时候会按照文件中的仓库源顺序查找下载。

  2. sources.list.d： 仓库目录，安装软件的时候会把该目录下的配置信息合并到 sources.list 中，关于查找顺序本人暂时还不清楚。

    我们把每个安装包的配置源单独放在 sources.list.d 目录中有以下好处：

    1. 文件名自带备注，方便查找。
    2. 添加或者删除只需要删除对应的文件就好，没有副作用。
    3. 方便第三方安装脚本使用。
